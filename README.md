he-ip6-cert-daily
=================

Simple script/utility for autosumbitting daily Hurricane Electric IPv6 certification tests

Usage
-----
* Install D compiler
* Just change username and password 
* Compile or run by *rdmd* as script 
* Put in cron daily + ~30mins

Deps
----
* traceroute
* dig
* whois
* ping
